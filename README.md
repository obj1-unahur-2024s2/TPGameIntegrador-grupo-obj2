## Reglas de Juego / Instrucciones
El objetivo del juego es derrotar a todos los enemigos que encontremos en el camino, para asi recolectar la mayor cantidad de puntos y poder llegar al portal de salida.

Enter => Atacar

Space => Saltar

W => Mover Arriba

A => Mover Izquierda

S => Mover Abajo

D => Mover Derecha
